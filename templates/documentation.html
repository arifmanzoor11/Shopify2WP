üìÑ Shopify to WordPress CSV Converter

This tool allows you to export product data from a Shopify store and convert it into a CSV file that is compatible with WordPress (WooCommerce).

üîß How It Works

  1. Upload JSON
    Export your Shopify products as a .json file or use the public Shopify API endpoint:
    Example: https://yourshop.myshopify.com/products.json
  2. Conversion
    The app processes each product and:
    • Detects if it's simple or variable
    • Creates a parent row (Type = variable) for products with multiple options
    • Generates variation rows (Type = variation) linked to the parent by SKU
    • Formats weights, prices, stock status, tags, categories, etc.
  3. Download CSV
    A WooCommerce-compatible CSV is downloaded, ready to be imported using:
    WooCommerce ‚Üí Products ‚Üí Import

üí° Input Options

  • Option 1: Upload a .json file containing Shopify product data
  • Option 2: Paste a public Shopify URL that ends in /products.json

üß† Notes

  • The app automatically renames the Colour attribute to Color
  • Missing SKU values are auto-generated using the product name
  • All prices, images, and weights are preserved
  • Only fields required by WooCommerce are included

üöÄ Getting Started

  1. Clone or download the project from GitHub (or your server)
  2. Install dependencies:
    pip install flask pandas requests
  3. Run the app:
    python app.py
  4. Visit:
    http://localhost:5000/
